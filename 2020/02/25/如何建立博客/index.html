如何建立博客

发表于 2020-02-25 | 分类于 hexo |
字数统计: 1.1k | 阅读时长 ≈ 4

博客的本地搭建

安装hexo

1.安装node.js
点击node.js下载,选择windows安装包，根据电脑的选择32位还是64位的，在这里我选择64位的安装

一路next即可。
2.安装git
点击git下载,选择最新的Windows版本

安装过程一直next就好。
3.测试node.js和git是否安装成功
在cmd在输入node -v、npm -v和git -v显示其版本号即为安装成功

4.安装hexo
安装Hexo很简单只需npm install hexo-cli -g即可，在cmd中或右键选择Git Bash Here进入来执行都可以

可以输入hexo -v检测hexo是否安装成功，如果能看到版本号安装成功

hexo本地预览

1.创建blog文件夹
在cmd中输入md blog，blog文件夹用来保存博客的所有内容
然后再输入cd blog，打开blog文件夹，cd blog在后面会经常用到，关于博文的创建与上传都会用到这个命令。
2.初始化博客
输入hexo init,出现“Start blogging with Hexo!”就说明，初始化成功。
3.本地预览
在cmd中输入hexo s,就会出现http://localhost:4000 .

复制链接到浏览器，当看到Hexo初始页面即可。

写博文

1.新建一篇博文
在cmd中执行hexo new “filename”(filename为文章名称)，执行后,在blog/source/_post下会出现一个Typora文件，打开就可以写自己的博文了。
2.写博文
具体参考Markdow教程
3.查看新博文
在写完博文后，可能会修改或添加博文内容，这时候就要下面的三步走
清除旧数据 hexo clean
生成新数据 hexo g
启动hexo hexo d

hexo博客乔迁Github

Github

1.注册Github
点击Github，选择右上角的Sign up注册

2.在Github下新建仓库

仓库名称必须写为username.github.io 其中username是你注册Github使用的用户名。
建好仓库后，在Settings中Github Pages这一部分要出现Your site is published athttps://username.github.io 字样,表明仓库具备了网页展示功能。

3.配置ssh密钥
在Git Bash Here输入ssh-keygen -t rsa -C “youremail@youremail.com“

执行这条命令会提示文件保存路径，可以直接按Enter即可,然后会在 .ssh 目录生产两个文件：id_rsa和id_rsa.pub。
用记事本打开.ssh目录下的id_rsa.pub文件，复制里面的内容，复制到Github中(点击右上角的头像->Settings->SSH and GPG keys->New SSH key)

点击Add SSH key即可。

添加图片

1.在添加图片的时候，我是直接打开用来写博文的Typaora，在文件->偏好设置-> 图像进行如下操作

这样是不能显示图片的，要在格式->图像->设置图片根目录，把目录设置到source就可以了。
2.采用七牛云添加图片
点击七牛云注册，注册好后进行邮箱实名认证，点击添加存储->空间管理->新建空间。

在空间管理->文件管理->上传文件,将图片上传上去即可

可在更多里面直接复制图片链接。(建议用七牛云比较好，图片加载速度快)

添加评论

  1. 添加评论必须要先选择一个主题。在这里我选择的是next主题，首先克隆项目到themes目录下，输入git clone git@github.com:iissnan/hexo-teme-next，在站点配置文件中把theme设置为next即可。注意：配置文件不是next下的配置文件，而是bolg下的配置文件。
  2. 下面就可以添加评论了，我选择的是leancloud作为评论插件，点击leancloud 注册,注册好后，先创建应用，直接默认即可。

    然后选择设置->安全中心，再应用 Keys中的AppID和AppKey复制到next下的配置文件，修改配置，定位到valine

    在安全中心中，将服务开关中打开数据存储，其余都关闭，在web安全域名输入你的博客链接点击保存。

    最后添加评论次数，设置->存储->创建Class，创建Comment来统计评论次数
